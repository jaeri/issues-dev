About This App

This app is designed for developers to track, document, and solve technical issues in projects.

It supports creating, editing, and managing issues in real-time, backed by Supabase and styled with PrimeNG.
Technologies Used
Angular 20

Structure based on traditional modules and components. Utilizes services, routing, and template-driven data binding.
PrimeNG 19

UI components like Card, Dialog, Skeleton, Sidebar, and more.
Supabase

PostgreSQL-based backend API for real-time database interaction.
App Architecture
Pages

View-based components such as List, Add, Delete, About.
Components

Reusable UI: Navbar, Form, Layout.
Services

Supabase API logic with async support.
Models

Interface-based types using TypeScript (Issue model).
UI Concepts Introduced
